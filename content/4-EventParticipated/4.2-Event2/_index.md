### Speaker Sessions

#### 1. Bảo Huỳnh – Docker: A containerization technology

This session helped me understand Docker and the role of containers in packaging and deploying applications. The talk highlighted Docker's benefits, including consistent runtime environments, easier deployment, scalability, and better support for DevOps and CI/CD workflows.

#### 2. Lê Hoàng Gia Đại – Combining AWS WAF with Machine Learning for Cyber Attack Detection

The speaker shared how AWS WAF can be combined with Machine Learning to improve cyber attack detection and web application protection. I learned more about analyzing abnormal traffic patterns and applying automation to strengthen application security.

#### 3. Nguyễn Quốc Bảo – Multiplayer in the Cloud: Connecting Godot Clients with AWS WebSockets

This session focused on building real-time connections for multiplayer games using AWS WebSockets. It helped me understand the client-server model, realtime communication, and how AWS can support low-latency applications.

#### 4. Trường – Technical sharing session

This sharing session broadened my view of how to approach a technical topic, present a solution, and connect cloud knowledge with real-world use cases. It also helped me improve the way I analyze problems and explain ideas clearly.

#### 5. Việt Phát – AWS Neptune for Building a Graph Knowledge Base for GraphRAG

The speaker introduced Amazon Neptune and how to build a Graph Knowledge Base for GraphRAG. I learned more about graph databases, relationships between entities, and how knowledge graphs can work with AI to improve information retrieval.

#### 6. Vinh Trần – From IT Helpdesk to Senior Sysadmin: Self-learning journey and career path

This session provided career guidance and showed the journey from IT Helpdesk to Senior Sysadmin. It emphasized the importance of self-learning, consistent practice, documenting lessons learned, and building a clear career roadmap.

### Results Achieved

- Gained practical knowledge about Docker, web security, WebSocket, GraphRAG, and system administration.
- Expanded my understanding of how AWS can be applied to different technical problems.
- Learned how speakers present technical ideas clearly with examples and practical applications.
- Gained more motivation to continue learning cloud skills and build a clearer professional development path.
