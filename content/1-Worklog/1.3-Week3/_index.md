# Worklog Week 3: VPC, Security Group, EC2 Windows/Linux, Snapshot and AMI

**Time:** 04/05/2026 - 08/05/2026

## 1. Weekly objectives

- Understood AWS networking by creating VPCs and planning subnets.
- Configured Security Groups to control inbound and outbound traffic.
- Launched and connected remotely to Windows and Linux EC2 instances.
- Practiced backup and optimization with Sysprep, EBS Snapshot and Custom AMI.
