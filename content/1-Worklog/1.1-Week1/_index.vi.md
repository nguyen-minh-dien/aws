# Worklog Tuần 1: Làm quen FCJ, AWS cơ bản, AWS Console và AWS CLI

**Thời gian thực hiện:** 20/04/2026 - 24/04/2026

## 1. Mục tiêu tuần

- Kết nối, làm quen với các thành viên trong cộng đồng First Cloud Journey (FCJ).
- Nắm vững kiến thức nền tảng về dịch vụ điện toán đám mây AWS.
- Thành thạo cách sử dụng AWS Management Console và AWS Command Line Interface (CLI).

## 3. Kết quả đạt được

- Hiểu rõ định nghĩa AWS và vai trò của các nhóm dịch vụ cốt lõi.
- Thành thạo thao tác trên giao diện Web Console để quản lý dịch vụ.
- Cài đặt thành công AWS CLI và thực hiện được các lệnh quản lý tài nguyên.
- Có khả năng tự khởi chạy và kết nối an toàn tới máy chủ ảo EC2.
