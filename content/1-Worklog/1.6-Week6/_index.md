# Worklog Week 6: Docker, Amazon ECR, Amazon ECS/Fargate and CI/CD

**Time:** 25/05/2026 - 29/05/2026

## 1. Weekly objectives

- Studied Docker Container architecture and application deployment process on AWS.
- Used Amazon ECR to store Docker images.
- Deployed systems with Amazon ECS and AWS Fargate.
- Understood CI/CD pipeline for automated application deployment.
- Practiced monitoring and cloud resource cleanup.

## 2. Implementation details

| Day | Work details | Date |

- Understood Docker packaging and AWS deployment workflow.
- Deployed Docker Containers on EC2 and stored images in Amazon ECR.
- Configured ECS Cluster, Task Definition, ECS Service and Application Load Balancer.
- Set up CI/CD Pipeline for automated build and deployment.
- Monitored system status with CloudWatch Container Insights.
